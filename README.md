# Калькулятор себестоимости 3D-печати

Интерактивный калькулятор на React и Next.js.

Учитывает:

- стоимость материала;
- время печати;
- электроэнергию;
- амортизацию принтера;
- подготовку и постобработку;
- резерв на брак;
- наценку.

## Онлайн-версия

https://ВАШ-ЛОГИН.github.io/3d-print-calculator/

Замените `ВАШ-ЛОГИН` на имя пользователя GitHub.

## Локальный запуск

Требуется Node.js 22 или новее.

npm ci
npm run dev
