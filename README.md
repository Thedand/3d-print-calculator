# Калькулятор себестоимости 3D-печати

Интерактивный калькулятор на React/Next.js. Учитывает материал, время печати,
электроэнергию, амортизацию, подготовку, постобработку, резерв на брак и наценку.

## Локальный запуск

Требуется Node.js 22.13 или новее.

```bash
npm ci
npm run dev
```

## Проверка production-сборки

```bash
npm run build
```

Для публикации установите зависимости, выполните production-сборку и используйте
платформу с поддержкой Node.js/Cloudflare Workers. Проект уже содержит настройки
для публикации через ChatGPT Sites.
